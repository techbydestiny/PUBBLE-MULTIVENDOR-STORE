# PUBBLE-MULTIVENDOR-STORE

A full-featured **multivendor e-commerce store** built with **Django**, **TailwindCSS**, and **HTML5**.  
This project enables multiple vendors to list and manage products, while customers can browse, add to cart, and checkout securely.  

---

## 🚀 Features
- 🔑 User authentication (signup, login, logout)  
- 🛍️ Vendor dashboard for product management  
- 📦 Add to cart & order management system  
- 💳 Secure checkout flow  
- 🎨 Modern, responsive UI built with TailwindCSS  
- ⚡ Fast, scalable Django backend  

---

## 🛠️ Tech Stack
- **Frontend:** TailwindCSS, HTML5  
- **Backend:** Django (Python)  
- **Database:** SQLite (default), easy to switch to PostgreSQL/MySQL  
- **Tools:** Git, Django ORM
